# Rollover Invoices

## Criteria

| **Field** | **Definition** |
| :--- | :--- |
| **Rollover** | Package that had rollovers. |
| **Rollover Type** | Type of Rollover: All, Pending, Completed. |

## Results

| **Column** | **Definition** |
| :--- | :--- |
| **Balance** | Balance due. |
| **Items Included** | Events included in package. |
| **Package Name** | Name of the Package. |
| **Seats** | Location of the seat. |
